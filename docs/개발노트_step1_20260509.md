## 프로젝트 개요

FastAPI 기반 웹 모니터링 시스템. DART, 금융위원회 데이터를 수집하여 키워드 매칭과 이메일 알람 제공.

**Step 1 목표:**
- FastAPI 기초 구조
- RSS 수집 파이프라인 (DART, FSC)
- 파일 기반 데이터 저장 (JSONL)
- 키워드 매칭 엔진
- 이메일 + 파일 알람

## Reviewer Conditions Applied (2026-05-09)

Reviewer가 APPROVED WITH CONDITIONS로 17개 Conditions + 4개 escalation을 반환했고, Architect가 escalation 결정을 내려서 Builder가 모든 항목을 적용했습니다.

### Architect 결정 (4건)

1. **DART endpoint** — `https://dart.fss.or.kr/api/rssFeeds.json` → `https://dart.fss.or.kr/api/todayRSS.xml` 로 교체 (config/sources.yaml).
2. **DART watchlist** — Step 1은 substring 매칭 유지. 시작 시 `logger.warning("DART watchlist using substring match — exact corp_code match deferred to Step 2")` 1회 출력.
3. **Pydantic V2 json_encoders** — `@field_serializer`로 즉시 마이그레이션 (`ExternalEvent.published_at/fetched_at`, `AlertLog.sent_at`, `SystemState.last_poll`). pytest deprecation 경고 0건 확인.
4. **KeywordRule.exclude_keywords** — 필드 유지. `# Step 2 scaffolding: matcher uses this in Step 2 for fuzzy/exclusion matching.` 주석 추가.

### 17 Conditions
