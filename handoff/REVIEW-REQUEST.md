## How This Works

Builder:
1. Self-review: what would Reviewer flag?
2. Verify: did the brief ship completely?
3. Test: what happens if things break?
4. Write this file
5. **Stop. Do not edit anything until Reviewer posts REVIEW-FEEDBACK.md**

Reviewer reads the diff first, then this file to verify Builder's claims.
